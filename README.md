##

Clone o repositório:
```bash
git clone https://github.com/brunomat480/gallery-plus.git
```

Baixe as dependências:
```bash
npm i
```
ou
```bash
yarn
```
ou
```bash
pnpm install
```

##

Inicie a aplicação:
```bash
npm run dev
```
ou 
```bash
yarn dev
```
ou
```bash
pnpm dev
```

##

Inicie o backend:
```bash
npm run dev-server
```
ou
```bash
yarn dev-server
```
ou
```bash
pnpm dev-server
```
